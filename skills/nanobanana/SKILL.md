description: Natural language image generation and manipulation. Use when the user asks about images in a general way and doesn't match a more specific image skill.
---

Use the nanobanana MCP server tools to help with image generation and manipulation based on the user's natural language request.

Analyze the request and pick the best tool:

- **generate_image**: Creating new images from text
- **edit_image**: Modifying an existing image file
- **restore_image**: Restoring or enhancing an old/damaged image
- **generate_icon**: App icons, favicons, UI elements
- **generate_pattern**: Seamless patterns, textures, backgrounds
- **generate_story**: Visual story sequences, tutorials, timelines
- **generate_diagram**: Flowcharts, architecture diagrams, schemas

User request: $ARGUMENTS
